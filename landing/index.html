Features

Built for the Shack

Azimuthal Map Generator
Generate high-resolution azimuthal projection maps centered on your QTH using NS6T online data. Instant PDF → PNG conversion.
Real‑time Compass Display
Live heading readout with a rendered azimuth compass on a Canvas overlay atop your generated map — always knows where your beam is pointing.
GS‑232 Serial Protocol
Talks directly to Yaesu G-600 and compatible rotors over USB serial using the standard GS-232 command set via the Web Serial API.
Simulation Mode
No hardware? No problem. A 50 ms software timer simulates azimuth movement. Explore and test the full UI without a rotor connected.
How It Works

From Grid to Heading

1
Enter Locator
Input your Maidenhead grid locator to set your station’s exact position on the map.
2
Generate Map
Fetch an azimuthal projection map from NS6T, rendered automatically for your coordinates.
3
Control Antenna
Click any heading or use presets to rotate your beam antenna with precision.
